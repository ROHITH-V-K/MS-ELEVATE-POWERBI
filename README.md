# MS-ELEVATE-POWERBI

Unemployment Rate Analysis in India
Project Overview

      This project analyzes unemployment trends in India using data visualization techniques. The dataset is obtained from the National Data & Analytics Platform (NDAP). The goal of this project is to understand unemployment patterns across different states, genders, age groups, and years through an interactive dashboard.

Objectives

    Analyze unemployment trends across different states in India

    Compare unemployment rates based on gender and age group

    Identify changes in unemployment across different years

    Present insights using interactive visualizations

Tools and Technologies

    Microsoft Power BI – Dashboard creation and visualization

    Microsoft Excel – Data preprocessing

    Dataset Source – National Data & Analytics Platform (NDAP)

Dataset

    The dataset includes the following attributes:

        State Name

        Year

        Gender

        Age Group

        Unemployment Rate (%)

Dashboard Features

      Average Unemployment Rate Card

      State-wise Unemployment Analysis

      Gender-based Unemployment Comparison

      Age Group Analysis

      Unemployment Trend Over Years

      Interactive Filters (Slicers)

Conclusion

The project provides a clear understanding of unemployment patterns in India through interactive visualizations. It highlights differences across states and demographic groups, helping users easily explore unemployment trends.
